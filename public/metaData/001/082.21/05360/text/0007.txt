स-सप्रे-टे-या शहाष्णव प्रयोगशील कलाकार
मि जगदीश कुलकर्णी — देवव्रत कुलकर्णी — नंदकिशोर डोंगरे … शरद अष्टपुवे - रविद्र मोडक तो मुकुंद कुलकर्णी … सुधीर भालेराव - द. रा.
वैद्य - श्रीकांत जोशी — राजेन्द्र दिडोरकर -— प्रदीप क्षत्रिय मैं-ब हरीष साखलकर च-चंद्रकांत वाणी …- प्रविण वैशंपायन - मोहीनी जोगलेकर
- शामा चिपजूणकर — वैजयंती गाडगील तो वनिता सहस्त्रबुर्द्ध — सुवर्णी साखलकर - मिलिंद V रानडे … विवेक गरुड - स्थिता लोंढे -

नंदकिशोर राऊत … धनेन्द्र अष्टपुवे — हर्षल अष्टपुवे - राजेश अभोणकर - किंर्ती मोडक - सुषमा भामरे … मदन शिप्रे - मीनल करकरे … सुवर्णी
केलकर तो देवयानी जहागीरदार … सुहास कुलकर्णी - साँदेपनाटकर-- अभय पंचाक्षरी- किरण इनामदार तो राजेश सेजपाल - जयश्री युगल
अंजली होना … मिलिंद कुलकर्णी -— श्री बाल - कैलास पाटील - संजय डबीर -_- सुनील देशपांडे - रविन्द्र कदम … अलका देशमुख
अजय शिवे … सुनिता ठोकछे - अविनाश पुरंदरे — किरण कुलकर्णी - शशा९क पेठे जा-ब रमेश पाठक जा-ब किशोर नीदुस्कर — ललित मासी … लक्ष्मण
सावजी - मिलिंद देशपांडे तो प्रसाद कोल्हटकर - सुनिल माजगावकर - अरुण गिते —- सुजाता पेठे — गीता कुलकर्णी व्य-विद्या पूंडे … शुभांगी
दिक्षीत - हेमा जोशी … प्रमोद तेंबे - सदानंद जोशी … रमेश मते - मेधा बेडेकर आ-बब विदचा दांडेकर - मंजिरी खक्वेंदवे - माधवी गाडगील
विजया कुलकर्णी - प्रेमानंद जाए … राजेन्द्र कुलकर्णी … भरत कुलकर्णी … उमेश पाचजाटील - उयोरुस्ता पंडीत … सुहासीनी कलकर्णी — विनोद
देशमुख … राजेद्र विभांडीक — राजेद्र शिरसाट — माधुही as? — रेवती जाधव — नयना राजे - सुप्रिया जोशी - राजश्री ढवठठे - दीपक वाछुजै
फान्सीस मौकढे -— विकास गरुड प्रकाश सदावतें — विकास जाधव - नरेद्र कुलकर्णी तो विजय जोशी - शशिकांत देशपांडे — दीलीप लोंढे
हेमंत टकले - हेमंत रिसबूड एकनाथ पाठक - उल्हास कुलकर्णी … जयंत वडनेरे - श्रद्धा लाले … सुशील लाले जा-ब निलेश देशपांडे … भाग्यश्री
देशपांडे - वैदेही साठे — रचना बेलगावकर … पंकजा कलमी तो पवित्रा कलंत्री तो वृषाली २र्वेलापूरकर - शेफाली गाडगील - केतन ठक्कर
आनंद . पंडीत .. शुभांगी कुलकर्णी — कल्पना सिंगासनें - मृणाल सोनवणे … दिलीप जहागिरदार - जयन्त निकम — भरत — बर्वे प्रमोद

देशपांडे - रमेश नेरकर - कैलास कौष्ठटी … प्रमोद कुलकर्णी - राजन अग्रवाल … गोविन्द कुलकर्णी तो मनोज पाटील - हिंरे’न देसासी … अजीत
मांडवगमें … शिरीष कुलकर्णी - सौ. चित्रा देशपांडे - सौ रेखा नाडगोडा — अरुण ढवलै - विवेक बेले … जितेन्द्र कुलकर्णी इत्यादी
